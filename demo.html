Raj Text Editor - Single File Solution

A complete WYSIWYG editor with all functionality in one JavaScript file

Quick Start

Include the JavaScript file and initialize the editor:

<!-- Include the editor script --> <script src="rich-text-editor.js"></script> <!-- Create a container --> <div id="my-editor"></div> <script> // Initialize the editor const editor = new RichTextEditor('my-editor', { height: '400px', theme: 'light', placeholder: 'Start typing...', autoSave: true }); </script>

Configuration Options

const editor = new RichTextEditor('container', { height: '400px', // Editor height theme: 'light', // 'light' or 'dark' placeholder: 'Type here...', // Placeholder text toolbar: 'full', // Toolbar configuration autoSave: true, // Enable auto-save spellCheck: true // Enable spell checking });

API Methods

Content Management

// Get/Set content const content = editor.getContent(); // Get HTML content editor.setContent('<p>Hello</p>'); // Set HTML content // Get plain text const text = editor.getText(); // Insert HTML editor.insertHTML('<strong>Bold text</strong>'); // Focus/Blur editor.focus(); editor.blur();

Document Operations

// Create new document editor.newDocument(); // Save document editor.saveDocument(); // Export in different formats editor.exportDocument('html'); // 'html', 'text', 'json'

View Modes

// Toggle view modes editor.toggleSourceMode(); // HTML source view editor.togglePreviewMode(); // Preview mode editor.toggleTheme(); // Switch theme editor.toggleFullscreen(); // Fullscreen mode

History Management

// Undo/Redo operations editor.undo(); editor.redo(); // Check if undo/redo available if (editor.canUndo()) { editor.undo(); }

Cleanup

// Destroy editor instance editor.destroy();

Auto-Initialization

You can also use data attributes for automatic initialization:

<div data-rich-editor data-height="300px" data-theme="dark" data-placeholder="Start writing..." data-auto-save="true"></div>

Integration Examples

ASP.NET Core

@{ ViewData["Title"] = "Rich Editor"; } <div class="editor-container"> <div id="content-editor"></div> <button type="button" onclick="saveContent()">Save</button> </div> <script src="~/js/rich-text-editor.js"></script> <script> const editor = new RichTextEditor('content-editor'); function saveContent() { const content = editor.getContent(); fetch('/api/content/save', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ content: content }) }); } </script>

React Component

import { useEffect, useRef } from 'react'; function RichEditor({ value, onChange }) { const editorRef = useRef(null); const instanceRef = useRef(null); useEffect(() => { instanceRef.current = new RichTextEditor(editorRef.current, { height: '400px' }); // Set initial value if (value) { instanceRef.current.setContent(value); } // Listen for changes instanceRef.current.editor.addEventListener('input', () => { onChange(instanceRef.current.getContent()); }); return () => { instanceRef.current?.destroy(); }; }, []); return <div ref={editorRef} />; }
